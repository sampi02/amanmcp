## Added

- **FEAT-UX3**: Backend Management automation in Makefile
  - `make install-ollama` - Install Ollama and pull default model
  - `make start-ollama` / `make stop-ollama` - Ollama server lifecycle
  - `make switch-backend-mlx` / `make switch-backend-ollama` - Easy backend switching
  - `make verify-install` - Verify installation and embedder connectivity
- `make install-local` now auto-configures PATH in shell config (no manual editing required)
- **Release automation**: `scripts/release.sh` now automatically updates all version references
  - Updates README.md version badge
  - Creates version changelog from unreleased.md
  - Updates CHANGELOG.md index
  - Resets unreleased.md to template
- Product version consistency check in `scripts/check-version-consistency.sh`
